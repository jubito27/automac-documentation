Introduction

Automac is a high-performance AutoML library designed to automate hyperparameter tuning using Optuna and Multithreading.

Why Automac? It handles the complexity of parallel execution and exhaustive search spaces so you can focus on the data.

Installation

Install the library directly from PyPI:

pip install automac

Quick Start

Get started with just a few lines of code:

from automac import AutoTuner

# Initialize with 50 trials per model
tuner = AutoTuner(n_trials=50)

# Run tuning for specific models
results = tuner.tune(model_keys=['xgb', 'rf', 'lgb'])

print(results)
